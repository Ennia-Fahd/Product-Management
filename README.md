# Product-Management
## Instructions 
#### -Install dependencies : npm install 
#### -Run the server : ng serve
#### -Run the local json server : json-server -w  Data/db.json -p 8089 (8089 is the port of the server)
## API :
### Conventions 
#### /products --> list all of products.
#### /products/id --> list a product by his id.
#### /users --> list all users.
#### /users/id --> list user by id
#### /products?_page=x_limit=y --> list all the products with pagination for the page x and with the size y.
